# CloudWatch Lite

CloudWatch Lite is a cloud-native full-stack platform being built to demonstrate modern software engineering, DevOps, and Site Reliability Engineering (SRE) practices using Laravel, Vue.js, Docker, Kubernetes, Terraform, CI/CD pipelines, monitoring tools, and cloud infrastructure concepts.

The project is designed as both a learning platform and a professional portfolio project, focusing on scalable architecture, observability, automation, and resilient deployments.

---

## Project Goals

* Build a scalable Laravel + Vue.js application
* Learn cloud-native development practices
* Implement containerised environments with Docker
* Explore Kubernetes orchestration
* Implement CI/CD pipelines using GitHub Actions
* Add monitoring and observability with Prometheus & Grafana
* Learn Infrastructure as Code using Terraform
* Implement authentication, RBAC, queues, and caching
* Simulate real-world production architecture and workflows
